# Push updates to EdInsights (GitHub)

From this repo, to push to **https://github.com/xd2164/EdInsights**:

```powershell
cd C:\Users\xiaoxuedu\community_citation_model
git add .
git commit -m "Your message"
git push edinsights main
```

Or in one line (PowerShell):

```powershell
cd C:\Users\xiaoxuedu\community_citation_model; git add .; git commit -m "Your message"; git push edinsights main
```

Replace `"Your message"` with a short description of the change.
